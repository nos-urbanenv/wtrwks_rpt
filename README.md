# wtrwks_rpt

3年Aセメスター、浄水処理実験課題のレポート作成用レポジトリ。

科学技術出版システム[Quarto](https://quarto.org/)を使用し、表組み・図作成を全て自動化している。そのため、実験データが差し替わった際にも簡単にレポートを再生成することができる。また、同じ分析を複数回繰り返すことが容易になるため、データのn数が増加した場合に対応がしやすい。今回は課題E(消毒実験)について、全班のデータを使って解析を行うことができた。

本当はすべての課題で全班のデータを使いたかった。（課題の作業に計画性がなく、時間切れで断念。）

## レポートの再現性について

Macであれば、以下の手順でレポートが再生成できる。Windowsの場合は、ファイルパスの指定方法が違うため、残念ながら再生成できない。この点については今後は、[fsパッケージ](https://fs.r-lib.org/)などを導入して対処していきたい。

1.  本レポジトリから以下のファイル・フォルダをローカルにダウンロードする。\
    report.qmd\
    /data\
    /data_raw\
    references.bib

2.  適当なディレクトリに上記の4つを格納する。

3.  RならびにRStudioを開き、report.qmdを開く。

4.  RStudio上部のRenderボタンを押すと、html版のレポートが再生成される。必要なパッケージがない場合には、install.packages()コマンドを利用してパッケージをインストールする。
